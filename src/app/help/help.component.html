Rejestracja nowego klienta

  1. Aby zarejestrować nowego klienta, użytkownik musi posiadać uprawnienia roli MANAGER

  2. Przechodzimy do zakładki Klienci

  3. Wybieramy zakładkę Dodaj, następnie wypełniamy formularz rejestracyjny

  4. Cały proces kończymy przyciskiem zarejestruj

 Rejestracja pacowników

  1. Aby zarejestrować nowego klienta, użytkownik musi posiadać uprawnienia roli MANAGER

  2. Przechodzimy do zakładki Pracownicy

  3. Wybieramy zakładkę Dodaj, następnie wypełniamy formularz rejestracyjny

  4. Cały proces kończymy przyciskiem zarejestruj

 Rejestracja kierowników

  1. Aby zarejestrować nowego klienta, użytkownik musi posiadać uprawnienia roli ADMINSTRATOR

  2. Przechodzimy do zakładki Kierownictwo

  3. Wybieramy zakładkę Dodaj, następnie wypełniamy formularz rejestracyjny

  4. Cały proces kończymy przyciskiem zarejestruj

 Przyjmowanie nowych zgłoszeń

  1. Aby dodać nowe zgłoszenie serwisowe należy posiadać uprawnienia roli MANAGER

  2. Przechodzimy do zakładki Zadania

  3. Wybieramy zakładkę Dodaj, przechodzimy przez listę kroków

  4. Cały proces kończymy przyciskiem zapisz zgłoszenie

  5. Ostatni krok to potwierdzenie przyjęcie sprzętu, należy je wydrukować po zapisie nowego zgłoszenia format PDF

Przydzielanie zadań

  1. Aby przydzielić zadanie do pracownika należy posiadać uprawnienia roli MANAGER

  2. Przechodzimy do zakładki Zadania

  3. Wybieramy zakładkę Przydziel, klikamy przycisk:

  4. Cały proces kończymy wyborem pracownika, należy zwrócić uwagę na obciążenie pracowników zadaniami

  5. Po wybraniu pracownika zadanie trafia do jego zakładki Zadania -> Twoje

Zatwierdzanie zadań

  1. Aby zatwierdzić zadanie należy posiadać uprawnienia roli MANAGER

  2. Przechodzimy do zakładki Zadania

  3. Wybieramy zakładkę Zatwierdź, klikamy przycisk: należy się upewnić czy wszystko zostało poprawnie uzupełnione możemy to sprawdzić poprzez kliknięcie:

  4. Po zatwierdzeniu zadania, status zmienia się na: OCZEKIWANIE NA ODBIÓR

Odbiór sprzętu

  1. Aby oddać sprzęt należy posiadać uprawnienia roli MANAGER klient powinien posiadać ze sobą potwierdzenie zostawienia sprzętu

  2. Przechodzimy do zakładki Klienci

  3. Wybieramy zakładkę Szukaj, szukamy naszego klienta (wyszukiwarka działa po wszystkich możliwych danych)

  4. Klikamy przycisk: zostanie nam wyświetlona lista zleceń konkretnego klienta

  5. Zlecenia są sortowane od najnowszych, wybieramy interesujace nas zlecenie

  6. Na końcu podsumowania zadania znajduje się przycisk klikamy go aby zakończyć zlecenie
Wirtualne stanowiska

  1. Wirtualne stanowiska są dostępne dla wszystkich ról, tylko MANAGER i ADMINISTRATOR może je dodawać, usuać, edytować

  2. Przechodzimy do zakładki Stanowiska -> Szukaj

  3. Mamy dostępne kafelki z wszystkimi stanowiskami dosępnymi w zakładzie pracy

  4. Klikając kafelkę lub możemy sprawdzisz szczegóły stanowiska:

    • Pracowników
    • Wyposażenie
 Dodawanie wyposażenia

  1. Aby dodać wyposażenie należy posiadać rolę MANAGER lub ADMINISTRATOR

  2. Przechodzimy do zakładki Wyposażenie -> Dodaj

  3. Wypełniamy formularz i kliamy przycisk Dodaj
